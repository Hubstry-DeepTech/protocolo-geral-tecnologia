## Ativar GitHub Pages

1. Vá em **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / pasta: **/ (root)**
4. Clique **Save**

O site estará disponível em: `https://hubstry-deeptech.github.io/protocolo-geral-tecnologia`
